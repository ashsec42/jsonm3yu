
# JSON → M3U Auto Generator

## Setup

1. Upload this repo to GitHub
2. Go to **Settings → Secrets → Actions**
3. Add:

JSON_URL = https://allinonereborn.online/airteltv-web/xchannels.json

4. Enable GitHub Actions
